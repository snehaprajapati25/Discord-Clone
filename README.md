# Discord Landing Page Clone

A modern Discord-inspired landing page built using HTML, CSS, and JavaScript. The project recreates Discord's UI with interactive components and smooth styling.

## 🚀 Features

- Fixed navigation bar
- Interactive navbar dropdown menus
- Hero section with background visuals
- Feature sections with videos
- Call-to-action (CTA) section
- Custom footer with language dropdown
- Social media icons using Font Awesome
- Custom scrollbar styling
- Hover effects and smooth UI interactions

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- Font Awesome
